# PrepVerse Education - UI/UX Design
# Overview
PrepVerse is a mobile-based educational platform UI designed to assist students in improving their aptitude, programming, and technical skills through interactive quizzes and performance tracking.
This project focuses solely on the UI/UX design and was created using Figma. The design emphasizes simplicity, accessibility, and an engaging learning experience.
